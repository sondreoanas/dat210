HTML ajax instructions:
	Include theese js files in header:
		<script src="static/js/mf_ajax.js"></script>
		<script src="static/js/mf_timeline.js"></script>
		<script src="static/js/nm_callbacks.js"></script>
		<script src="static/js/templater.js"></script>

	To fill an element with html content:
		<div data-fill="/getHTML?html=someContent.html"></div>
	
	
	To replace and element with html content:
		<div data-replace="/getHTML?html=someContent.html"></div>
	
	
	To make a button fill element with html content:
		<button data-target="toBeFilled" data-fill="/getHTML?html=someContent.html">Button that will fill the h1</button>
		<h1 id="toBeFilled">
			Some h1 to be filled
		</h1>
	
	To make a button replace element with html content:
		<button data-target="toBeReplaced" data-fill="/getHTML?html=someContent.html">Button that will replace the h1</button>
		<h1 id="toBeReplaced">
			Some h1 to be replaced
		</h1>
	
	To make a button place html content before or after an other element:
		before:
			<button data-target="toBePreceded" data-before="/getHTML?html=mf_sucsess.html">Some button to create element to precede the first h1</button>
			<h1 id="toBePreceded">
				Some h1 to be preceded
			</h1>
		after:
			<button data-target="toBeFollowed" data-after="/getHTML?html=mf_sucsess.html">Some button to create element to follow the second h1</button>
			<h1 id="toBeFollowed">
				Some h1 to be followed
			</h1>
	
	To make a button submit a form and recieve a response
		
		In an html file:
			<form id="someForm" action="/handleLogin">
				<input type="text" name="email" id="email"><br>
				<input type="text" name="password" id="password"></br>
			</form>
			<button data-formid="someForm" data-callback="callBackExample">Hello there</button>
		
		In static/js/nm_callbacks.js:
			function callBackExample(response){ // responce is an object recieved from python
				// to fill an element
				mf_ajaxHandler.fillElement(elementid = "someId", url = "/getHTML?html=someFile.html");
				// to replace an element
				mf_ajaxHandler.replaceElement(elementid = "someOtherId", url = "/getHTML?html=someOtherFile.html");
				// to place before
				mf_ajaxHandler.placeBeforeElement(elementid = "someOtherId", url = "/getHTML?html=someOtherFile.html");
				// to place after
				mf_ajaxHandler.placeAfterElement(elementid = "someOtherId", url = "/getHTML?html=someOtherFile.html");
			}
